JYMAG - Capabilities' window



This window allows checking the phone's capabilities.

The port and its settings are the ones set in the main window.

  1. Choose the capabilities you want to check in the upper-left side of the window
  2. Press "Get selected capabilities"
  3. The selected phone capabilities show up in the text area below
  4. Press "Exit" to close the window
  5. Use the font size spinner to adjust the font size of this window
